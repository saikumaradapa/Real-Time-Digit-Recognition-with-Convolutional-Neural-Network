Hand Written Digit Recognizer

by Sai Kumar Adapa

Project Title: Real-time Digit Recognition System
Developer Name: Sai Kumar Adapa
Tech Stack: Flask, Keras, OpenCV, HTML, Bootstrap
Description: A real-time digit recognition system that captures video from a device's camera, processes the stream frame-by-frame, and predicts the handwritten digit present in a designated rectangular region on the screen. Leveraging the power of deep learning, the system predicts the digit and displays the result, all in real time.
